# @trop/diary_cli

Client Command Line for interacting with `@trop/diary_api` - A Logging Service

## Usage

```bash
trop-diary set-endpoint http://api-endpoint-somewhere.com
trop-diary login root@mail.com
trop-diary message-list
```
